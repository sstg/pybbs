== 特性

- 社区兼容性（IE9+）
- 页面自适应布局
- 本地登录，注册
- 登录时有验证码，尝试登录次数的限制
- 使用 `Spring-Boot` 开发
- 权限使用 `Spring-Security` ，基于url做的权限方便配置管理
- 使用的 `Spring-JPA` 操作数据存储
- 使用 https://github.com/lepture/editor[editor] 作为 Markdown编辑器, 书写更方便, 还支持截图粘贴上传
- 还增加了 https://github.com/wangfupeng1988/wangEditor[wangEditor] 作为富文本编辑器
